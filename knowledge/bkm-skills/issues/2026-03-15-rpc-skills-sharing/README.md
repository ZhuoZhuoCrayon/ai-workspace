# 观测经验 Skills 化 — 将 RPC 服务排障知识固化为 AI 的专家能力

## 0x01 背景

### a. Why

bk-rpc-inspection Skill 已积累了从知识层、工具层、原子技能层到工作流层的完整 RPC 排障能力。这套实践值得对外分享，帮助团队了解如何将观测经验系统性地固化为 AI 可执行的专家技能。

### b. 目标

1. **技术文章**：输出一篇完整的技术分享文章，涵盖设计理念、四层架构、核心知识沉淀和实战案例
2. **分享 PPT**：基于技术文章制作演讲用 PPT，适合团队内部或技术社区分享

## 0x02 实现路线

### a. 建议的方案

1. 基于 bk-rpc-inspection Skill 源码（SKILL.md + references/）提炼核心内容
2. 参照已有参考文档（`references/` 目录下的草稿）打磨技术文章
3. 基于文章内容和 PPT 模板制作分享 PPT

### b. 约束

- bk-rpc-inspection Skill 仍在持续迭代中，文章内容需基于最新 Skill 源码
- PPT 框架和样式参考模板文件（`references/ppt-template.pptx`）
